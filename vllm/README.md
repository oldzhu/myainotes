## Bilingual policy

All documents in this notes repo must have both English and Simplified Chinese versions.
For every English Markdown file, create a matching `*.zh-CN.md` file, and include cross-links
at the top of both files so readers can switch languages.

Validate with:

```bash
make bilingual-check
```

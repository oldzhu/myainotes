## 双语规范

此笔记仓库中的所有文档必须同时提供英文版与简体中文版。
每个英文 Markdown 文件都应有对应的 `*.zh-CN.md` 文件，并在两个文件顶部添加
互相切换的链接，方便读者切换语言。

可用以下命令校验：

```bash
make bilingual-check
```
